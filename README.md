# Sales Analytics Dashboard

Interactive Excel dashboard created to analyze sales performance, profit trends and business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Average Sales KPI
- Category Wise Sales
- City Wise Sales Analysis
- Payment Method Analysis
- Dynamic Slicers

## Key Insights
- Electronics category generated the highest sales.
- Delhi achieved the highest revenue.
- UPI was the most used payment method.
- Sales performance increased in later months.
